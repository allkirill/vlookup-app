VLOOKUP-APP
Макрос для Excel, который автоматизирует обновление цен и остатков по прайс-листам поставщиков. Работает локально, не требует установки, ничего не отправляет в интернет.

Что это такое?
Это решение для небольших интернет-магазинов, которые устали обновлять цены вручную. Стандартный ВПР (VLOOKUP) не справляется с кривыми артикулами, пробелами и смешанными языками. Этот макрос решает и автоматизирует решение многих проблем.

Основные возможности:
ВПР по двум артикулам — если не нашёл по артикулу, ищет по штрих-коду.
Нормализация — заменяет русские буквы на английские (А→A, В→B), убирает лишние пробелы и невидимые символы.
Суффиксы — можно защитить свои наценки от перезаписи.
Бонус: Поиск новинок — отдельный макрос для поиска товаров, которых ещё нет на сайте.

Как использовать
Скачай файл vlookup-app.xlsm из раздела Releases (или прямо из репозитория).
Открой файл в Excel (2016 или новее).
Разреши выполнение макросов (Excel спросит при открытии).
Настрой лист «Настройки» под свои столбцы.
Запусти макросы кнопками на панели.

Три кнопки — три шага:
Загрузить данные — выбрать файл выгрузки сайта и прайс поставщика.
Обновить — макрос найдёт совпадения и обновит цены/остатки.
Сохранить — выгрузить результат в файл для импорта на сайт.

Безопасность
Локальная работа. Макрос работает только на твоём компьютере.
Нет интернета. Никакие данные о ценах и артикулах не уходят на внешние серверы.
Код под паролем. Защищено от случайных изменений.
Как проверить безопасность — подробно расписано на сайте проекта.
